# 📘 PDF Knowledge Base using Agno + Groq + LanceDB

An AI-powered Retrieval-Augmented Generation (RAG) assistant built using Agno framework, Groq LLMs, LanceDB vector database, and HuggingFace embeddings.

This project allows users to:
- Load PDF documents into a vector database
- Perform semantic search on PDF content
- Ask natural language questions
- Retrieve context-aware answers using Groq LLMs
- Enhance responses with web search when needed

---

# 🚀 Features

- ✅ PDF Knowledge Base Integration
- ✅ Retrieval-Augmented Generation (RAG)
- ✅ LanceDB Vector Storage
- ✅ HuggingFace Embeddings
- ✅ Groq LLM Integration
- ✅ Web Search using DuckDuckGo
- ✅ Hybrid Search Support
- ✅ Streaming Responses
